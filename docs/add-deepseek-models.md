# Add DeepSeek Local Models

Default language: English. Chinese version: [add-deepseek-models.zh-CN.md](add-deepseek-models.zh-CN.md).

Use [the English prompt](../prompts/add-deepseek-models.md) to add local catalog entries for:

- `deepseek-v4-flash`
- `deepseek-v4-pro`

Key points:

- Back up files before editing.
- Reuse the existing provider.
- Do not change `model_provider`, provider sections, `base_url`, or tokens.
- Update `models_cache.json`.
- Verify with `codex debug models` or an equivalent command.
